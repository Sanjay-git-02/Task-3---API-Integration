# 🚀 DevLaunch Hub

DevLaunch Hub is a responsive web application that brings together four interactive mini-projects into a single platform. Built using HTML5, CSS3, and JavaScript, it demonstrates API integration, DOM manipulation, asynchronous programming, and responsive web design.

This project was developed as part of my **Full Stack Web Development Internship**.

---

## 📖 Overview

DevLaunch Hub provides quick access to four useful web applications from a single dashboard:

- 🌦️ Weather Dashboard
- 🎬 Movie Search Hub
- 💬 Quote Generator
- 👤 Random User Generator

Each module is independently functional while maintaining a consistent user interface and user experience.

---

## ✨ Features

### 🌦️ Weather Dashboard
- Search weather by city
- Real-time temperature
- Humidity and wind speed
- Weather condition icons
- Error handling for invalid locations

### 🎬 Movie Search Hub
- Search movies instantly
- View posters
- IMDb ratings
- Genre and release year
- Movie plot summary

### 💬 Quote Generator
- Generate random motivational quotes
- Display author information
- Refresh quotes instantly

### 👤 Random User Generator
- Generate random user profiles
- Display profile image
- Name, email, phone number
- Location details

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Fetch API
- REST APIs

---

## 🌐 APIs Used

### Weather Dashboard
- **OpenWeatherMap API**
- Provides live weather information.

### Movie Search Hub
- **OMDb API**
- Retrieves movie information and posters.

### Quote Generator
- **Quotable API / ZenQuotes API**
- Generates random inspirational quotes.

### Random User Generator
- **RandomUser API**
- Generates random user profiles.
